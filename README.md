# Finished?

Monitor Blender renders from Telegram.

Start a render in Blender, leave the computer, and check its progress from your phone. Finished? shows when the render starts, tracks its progress, estimates the remaining time, and lets you know when it is finished, failed, or cancelled.

## Features

- Telegram notifications for render start, completion, cancellation, and errors
- Live render progress tracking
- Estimated time remaining (ETA)
- Average frame render time
- Stalled render detection
- One live status message that updates as the render progresses
- Detailed render monitoring through a Telegram Mini App
- Interactive render duration graph
- Read-only share links
- Support for image sequence rendering workflows
- Simple device pairing through Telegram

## How to use

1. Install the Finished? add-on in Blender.
2. Open the [Finished? Telegram bot](https://t.me/render_finished_bot) and connect your Blender device with a pairing code.
3. Choose **Render → Render Animation with Finished?** in Blender.
4. Follow the render status in Telegram.

## Telegram Monitoring

Finished? keeps render information organized in a single status message:

- Current frame progress
- Completion percentage
- Elapsed time
- Estimated remaining time
- Average frame render time

When the render finishes, you'll receive a final notification with the result.

## Telegram Mini App

The built-in Mini App provides a mobile-friendly view of your active render:

- Live progress visualization
- Render duration graph
- Approximate finish time
- Recent render history
- Lifetime statistics

Everything is accessible directly from Telegram.

## Supported Render Formats

Finished? supports Blender animation renders saved as image sequences, including PNG, JPEG, TIFF, and OpenEXR.

## Designed For

- Motion Designers
- Blender Artists
- Freelancers
- Small Studios

## Installation

Download the latest release and install the ZIP file through:

`Edit → Preferences → Add-ons → Install`

After installation, connect your Telegram account and start monitoring renders.
